# 🌍 IP Trace Tool

A Python tool that traces IP addresses and retrieves geolocation information from multiple online sources.

## Features

- IP Address Lookup
- City Detection
- Region Detection
- Country Detection
- ISP / Organization Information
- Latitude & Longitude Coordinates
- Google Maps Location Link
- Multiple API Fallback Support

## Technologies Used

- Python 3
- Requests Library
- ipinfo.io API
- ip-api.com API
- ipwhois.app API

## Installation

```bash
git clone https://github.com/yourusername/ip-trace-tool-python.git
cd ip-trace-tool-python
pip install -r requirements.txt
python best_ip_trace_tool.py
```

## Usage

```text
📝 Enter the IP address to trace: 8.8.8.8

🔍 Tracing IP, please wait...

📊 IP Trace Report
---------------------------------------
IP Address: 8.8.8.8
City: Mountain View
Region: California
Country: United States
Location: 37.4056,-122.0775
Organization: Google LLC
```
